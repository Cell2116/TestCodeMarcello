# TestCodeMarcello
PT Inovasi Daya Solusi

Author: Marcello Octavyo Anugrahanto

Tech - tools:
- Package Manager : NPM 
- Frontend : Vue.js, axios.js 
- Backend : Node.js, Express.js, postman
- Database : PostgreSQL

Library:
- Backend : pg, express, body-parser
- Frontend : axios

Cara menjalankan sistem:
- Diperlukan install NPM pada Backend & Frontend
- Install library pada directory masing-masing
- Run Frontend dengan "npm run dev"
- Run Backend dengan "node app.js"
